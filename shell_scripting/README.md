This lab covers the basics of shell scripting, with a focus on Bash. Shell scripts allow us to automate repetitive tasks and manage files efficiently.

Here 'backup.sh' is a shell script that automates file/directory creation and copying for files with the .c/.h extensions.

To run (Type the following commands in a terminal):
chmod +x backup.sh
./backup.sh test

To see the new clone directory of 'shell/scripting' called 'test', run:
cd shell_scripting/test 

To see the 3 files: 'a.c', 'a.h' and 'b.c' created cloned in test, run:
ls
